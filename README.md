Welcome to [ML Modeling](https://fluxpuls.github.io/ML-modeling/) - an interactive platform for exploring and experimenting with machine learning models directly in your browser. No servers, no API keys, and all data stays private. 

## How to Use

- Free in-browser ML Picker - interactive AI chat that helps you choose the right ML model for your problem. It runs directly in your browser, offering an easy way to experiment with different ML models without needing any setup. Powered by prompt engineering, it provides a conversational experience to guide you through tasks and challenges. You can use ChatGPT or your preferred chat model to explore and learn.
- Dive deeper into the math of ML algorithms -Want to understand the theory behind the models? Check out the accompanying book (the introduction is available in the Jupyter notebook)
